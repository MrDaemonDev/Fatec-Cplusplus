# FATEC — Exercícios e Aprendizagem em C++

Repositório criado para organizar **exercícios e códigos de estudo em C++** desenvolvidos durante o curso na **FATEC**.

O objetivo é manter o conteúdo das aulas separado dos exercícios práticos, facilitando revisão e evolução ao longo das disciplinas.

---

# Estrutura do Repositório

```text
fatec
 ├─ Aprendizagem
 │   ├─ code01.cpp
 │   ├─ code02.cpp
 │   ├─ code03.cpp
 │   └─ ...
 │
 └─ Exercicios
     ├─ L01A.cpp
     ├─ L01B.cpp
     ├─ L01C.cpp
     └─ ...
```

---

# Pasta `Aprendizagem`

Contém exemplos e códigos feitos durante o processo de aprendizado.

Padrão de nome:

```text
code01.cpp
code02.cpp
code03.cpp
```

Uso:

* testes de conceitos
* exemplos mostrados em aula
* pequenos experimentos

---

# Pasta `Exercicios`

Contém listas de exercícios resolvidas.

Padrão de nome:

```text
L01A.cpp
L01B.cpp
L01C.cpp
```

Onde:

* **L01** → Lista / fase 01
* **A, B, C...** → exercício da lista

Exemplo:

```text
L01A.cpp → Exercício A da lista 01
L01B.cpp → Exercício B da lista 01
```

---

# Próximas Listas

Quando iniciar uma nova lista de exercícios, o padrão muda.

Exemplo:

```text
L02A.cpp
L02B.cpp
L02C.cpp
```

Onde:

* **L02** → Lista 02
* **A, B, C...** → exercícios da lista

---

# Objetivo do Repositório

* Organizar exercícios da faculdade
* Registrar progresso no aprendizado
* Servir como material de revisão
* Praticar lógica de programação em C++

Repositório criado para estudos no curso da **FATEC**.
